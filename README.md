# 🚀 Three.js Learning Journey

Welcome to my **Three.js Learning Repository**! 👋

This repository documents my journey of learning **Three.js** from the basics to implementing real-world 3D models. Each phase focuses on a different core concept of Three.js, helping build a strong foundation in WebGL and interactive 3D graphics.

---

## 📚 Learning Phases

### 📘 Phase 1: Introduction to Three.js

* Understanding the Three.js ecosystem
* Setting up a basic Three.js project
* Creating a Scene
* Creating a Camera
* Setting up a WebGL Renderer
* Rendering the first 3D object

---

### 📘 Phase 2: Camera Controls (OrbitControls)

* Camera rotation
* Zooming
* Smooth damping
* Auto rotation
* Camera constraints
* OrbitControls configuration and methods

**Key Concepts**

* `enableDamping`
* `enableZoom`
* `autoRotate`
* `target`
* `minDistance` / `maxDistance`
* `minPolarAngle` / `maxPolarAngle`
* `saveState()`
* `reset()`
* `update()`

---

### 📘 Phase 3: Geometries

Explored various built-in geometries available in Three.js.

**Geometries Covered**

* 🔵 SphereGeometry
* 🟢 CapsuleGeometry
* 🟡 TorusGeometry
* 🔴 TorusKnotGeometry

Also learned about:

* Vertices
* Faces
* Geometry segments
* Performance considerations

---

### 📘 Phase 4: Lights, Materials, Textures & GUI

#### 💡 Lights

* Hemisphere Light
* Directional Light
* Point Light
* Spot Light

#### 🎨 Materials

* MeshBasicMaterial
* MeshLambertMaterial
* MeshPhongMaterial
* MeshStandardMaterial
* MeshPhysicalMaterial

#### 🖼️ Textures

* Color Map
* Normal Map

* Displacement Map

#### 🎛️ Lil GUI

* Material Controls
* Mesh Controls
* Light Controls

#### 🎮 OrbitControls

* Smooth interaction
* Zoom
* Rotation
* Damping

---

### 📘 Phase 5: 3D Model Implementation

Implemented a real-world **GLB Car Model**.

Topics covered:

* GLTFLoader
* Model Loading
* Auto Scaling
* Model Centering
* HDR Environment Maps
* Ambient & Directional Lighting
* OrbitControls
* Animation Loop
* Responsive Rendering
